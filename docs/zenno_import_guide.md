## Подготовка к импорту

### 1. Подготовка профиля ZennoPoster
- Убедитесь, что профиль ZennoPoster полностью настроен:
  - Установлены все необходимые расширения
  - Выполнен вход во все аккаунты
  - Настроены все необходимые параметры
- Закройте все окна Chrome и ZennoPoster
- Найдите папку с профилями ZennoPoster (обычно находится в `%USERPROFILE%/Documents/ZennoLab/ZennoPoster/Profiles`)

### 2. Подготовка Chrome Profiles Manager
- Убедитесь, что программа закрыта
- Проверьте наличие папки `data/profiles` в корне проекта
- Если папка не существует, создайте её

## Структура профиля Chrome

Для успешного импорта профиль должен содержать следующие папки и файлы:

```
Profile {name}/
├── Extensions/                    # Расширения Chrome
├── Local Extension Settings/      # Локальные настройки расширений
├── Sync Extension Settings/       # Синхронизированные настройки
├── Extension Rules/               # Правила расширений
├── Extension Scripts/             # Скрипты расширений
├── Extension State/               # Состояние расширений
├── Local Storage/                 # Локальное хранилище
├── IndexedDB/                     # База данных IndexedDB
├── Session Storage/               # Хранилище сессий
├── Network/                       # Сетевые настройки
├── Cache/                         # Кэш браузера
├── Code Cache/                    # Кэш кода
├── GPUCache/                      # Кэш GPU
├── Preferences                    # Настройки профиля
├── Secure Preferences            # Безопасные настройки
└── Web Data                      # Данные веб-сайтов
```

## Процесс импорта

### 1. Запуск программы
```bash
python main_gui.py
```

### 2. Импорт профиля
1. В главном меню нажмите "📥 Импорт из ZennoPoster"
2. В открывшемся окне нажмите "Выбрать папку"
3. Укажите путь к папке с профилями ZennoPoster
4. Выберите нужные профили из списка
5. Нажмите "Импортировать выбранные"

### 3. Что происходит при импорте
1. Создается новая папка профиля в `data/profiles`
2. Создаются все необходимые подпапки
3. Копируются файлы из папки Default профиля ZennoPoster
4. Копируются остальные файлы и папки
5. Добавляется комментарий "Импортирован из ZennoPoster"

## Проверка импорта

После импорта необходимо проверить:

1. **Наличие профиля**
   - В меню "📖 Просмотр профилей" должен отображаться импортированный профиль
   - Комментарий должен быть "Импортирован из ZennoPoster"

2. **Запуск профиля**
   - В меню "🚀 Запуск профилей" выберите импортированный профиль
   - Профиль должен запуститься без ошибок
   - Все расширения должны быть на месте

3. **Проверка расширений**
   - В меню "🧩 Работа с расширениями" выберите импортированный профиль
   - Должны отображаться все установленные расширения
   - Расширения должны быть активны и работоспособны

## Возможные проблемы и их решение

### 1. Профиль не запускается
- **Проблема**: Ошибка "не удается найти указанный путь"
  - **Решение**: Проверьте наличие всех необходимых папок в профиле
  - **Решение**: Попробуйте импортировать профиль заново

### 2. Расширения не работают
- **Проблема**: Расширения не отображаются или не активны
  - **Решение**: Проверьте папку Extensions в профиле
  - **Решение**: Проверьте папки настроек расширений
  - **Решение**: Попробуйте переустановить расширения через меню "Работа с расширениями"

### 3. Не сохраняются настройки
- **Проблема**: После перезапуска сбрасываются настройки
  - **Решение**: Проверьте права доступа к папке профиля
  - **Решение**: Проверьте наличие и содержимое файла Preferences

### 4. Ошибки при импорте
- **Проблема**: Импорт завершается с ошибкой
  - **Решение**: Проверьте путь к профилям ZennoPoster
  - **Решение**: Убедитесь, что все процессы Chrome закрыты
  - **Решение**: Проверьте права доступа к папкам

## Советы

1. Всегда закрывайте Chrome перед импортом профилей
2. Делайте резервные копии важных профилей
3. Проверяйте работоспособность после импорта
4. Следите за логами при возникновении проблем

## Полезные команды

1. Убить все процессы Chrome:
```bash
taskkill /F /IM chrome.exe
```

2. Очистить кэш профиля:
```bash
# Удалить содержимое папок
Cache/
Code Cache/
GPUCache/
```

3. Проверить наличие всех папок:
```bash
dir "Profile {name}" /s /b
```
